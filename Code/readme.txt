This is a readme file for homework 3.

1. To run the main file,use 
    python3 main.py

2. To run only one test case in the homework2, use function
    runcase_with_id(i)

    Here i varies from 0-9. 

3. All results will be saved in the subdirectory "./result", and you can also find the latex code for generating the tables there.

4. To change the options for convergence, use 
    Obj.converge_options.ftol = (some number you want)

    gtol, xtol can also be set in a similar way.
